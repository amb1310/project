# How to Deploy on Streamlit Cloud

1. Go to https://streamlit.io/cloud and sign up with your GitHub account.
2. Create a new GitHub repository and upload all the files from this folder.
3. In Streamlit Cloud, click "New app" and select your GitHub repo.
4. Set the main file to: app.py
